    WINDOWS OS清除DNS缓存的方式

    步骤一，在桌面左下点击WINDOWS图示

    • 步骤二，输入cmd后，按下Enter

    • 步骤三，会出现系统管理员，请于提示符号下输入ipconfig / flushdns后按下

    • 步骤四，待出现[成功清除DNS解读器缓存]字样，表示成功。

    • MAC OS清除DNS缓存的方式

    步骤一，点选Finder桌面菜单前往/实用工具。

    • 步骤二，点击“终端.APP”

    • 步骤三，会出现系统管理员，请于提示符号下输入dscacheutil -flushcache后按下Enter，即完成设定。

    Windows XP设置DNS服务器

    步骤一，先点选[我的计算机]

    • 步骤二，「网上邻居」图标上单击鼠标右键，再点选“属性

    • 步骤三，此页面会列出你所有的网络卡和联机设定

    • 步骤四，如果你是透过路由器或是DHCP联机的，请在接网络线的适配卡右键单击并点选内容，至于PPPoE的等一下会提到

    • 步骤五，在内容可以看到各种网络功能，选取TCP / IP之后点一下属性

    • 步骤六，进入IP和DNS的设定窗口，把DNS改成手动输入，填入你DNS服务器IP，确定之后就完成了惯用DNS服务器使用：8.8.8.8备用DNS服务器使用：8.8.4.4

    • 步骤七，PPPoE（拨号上网）的用户，请回到适配卡设定，对着拨号设定右键单击>>内容，切换至“网络功能”卷标，选取“IPv4”，然后“属性”

    • 步骤八，把DNS改成手动输入，填入IP地址，确定之后不会马上完成设定，要重新拨号时才会议用。首选DNS服务器使用：8.8.8.8

    • Windows 7和8设置DNS服务器

    步骤一，在工具栏，网络按右键

    • 步骤二，接着点击“开启网络和共享中心”

    • 步骤三，在网络和共享中心，点击左边的 “更改适配器设置”

    • 步骤四，画面会列出你所有的网络卡和联机设定

    • 步骤五，如果你是透过路由器或是DHCP连线的，请在接网路线的适配卡按右键并点选属性，至于PPPoE的等一下会提到

    • 步骤六，在内容可以看到各种网路功能，选取的IPv4之后点一下属性

    • 步骤七，进入IP和DNS的设定窗口，把DNS改成手动输入，填入你DNS服务器IP，确定之后就完成了首选DNS服务器使用：8.8.8.8备用DNS服务器使用：8.8.4.4

    • 步骤八，PPPoE（拨号上网）的用户，请回到适配卡设定，对着拨号设定按右键>>属性，切换至“网络”标签，一样的，选取IPv4，然后属性

    • 步骤九，把DNS改成手动输入，填入你DNS服务器IP，确定之后就完成了，首选DNS服务器使用：8.8.8.8备用DNS服务器使用：8.8.4.4

    • Mac OS X设置DNS服务器

    步骤一，首先打开「系统偏好设定」，选择「网络」。

    • 步骤二，选择自己所使用的网络，然后开启「进阶」的选项。

    • 步骤三，选择「DNS」页签，在「DNS服务器」字段中新增这两台DNS服务器IP地址就可以了。
